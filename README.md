# SocialVotes機能まとめ
## SV看板の作り方
看板に下記記入
```
１行目　vote
２行目　名前
```
## コマンド一覧
```
正式名称　 /socialvotes
エイリアス /sv
```
### ====誰でも可====
### `/sv setgroup <name>`
- 投票グループを作成
### `/sv add <name>`
- クリックでSV看板をグループに追加できる状態になる
- SV看板以外をクリックで状態解除
### `/sv list (<name>)`
- グループ一覧表示
- グループ名追記の場合、SV看板一覧表示
### `/svtp <id>`
- 指定IDのSV看板へテレポート
### `/svupdate`
- クリックした看板を正式なSV看板にする状態になる
- SV看板以外をクリックで状態解除
### ====グループ作成者とOPのみ可====
### `/sv remove`
- クリックでSV看板をグループに除外できる状態になる
- SV看板以外をクリックで状態解除
### ====OPのみ可====
### `/sv delhere`
- 実行者の足元のSV看板を消去
### `/sv allclear <name>`
- グループと属するSV看板を一括消去
## イベント時の投票システム作成ガイド
### ①投票先のプレイヤーがSV看板を立てる
### ②主催者は投票グループを作成してSV看板を登録する
### ③Shift右クリックから詳細設定GUIで各設定をする
### ④投票期間終了後、投票結果GUIにて結果確認
